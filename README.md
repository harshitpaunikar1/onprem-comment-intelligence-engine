# On-Prem Comment Intelligence Engine

> **Domain:** Marketing / AdTech

## Overview

Private, always-on assistant for comment streams using local Gemma-31B with retrieval-augmented generation for summarization and question-answering without external data exposure.

## Skills & Technologies

- Local LLM Hosting
- Retrieval-Augmented Generation
- Text Summarisation
- Docker Containerization
- Heroku Deployment
